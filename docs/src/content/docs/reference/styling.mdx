## CSS custom properties

| Property | Default | Description |
|---|---|---|
| `--audio-visualizer-color` | `currentColor` | Bar colour. Takes priority over the `color` attribute. Falls back to the inherited CSS `color` property when unset. |
| `--audio-visualizer-transition` | `100ms` | Duration of bar height transitions. Lower = snappier; higher = smoother. |

## Three ways to set colour

### 1. CSS custom property (recommended)

Keeps colour in CSS — works with design tokens and theme switching.

```css
/* Stylesheet */
audio-visualizer {
  --audio-visualizer-color: #6366f1;
}

/* Or inline */
```

```html
<audio-visualizer style="--audio-visualizer-color: #6366f1"></audio-visualizer>
```

### 2. CSS inheritance

The bars use `background-color: currentColor` internally. Set `color` on any ancestor and the bars inherit it — no attribute or custom property needed.

```html
<div style="color: #ec4899">
  <audio-visualizer></audio-visualizer>
</div>
```

<Aside type="tip">
Inheritance is great for buttons or toolbars where the surrounding text color already matches your brand. Zero extra code.
</Aside>

### 3. HTML `color` attribute

A convenience attribute for quick, one-off colour overrides. The `--audio-visualizer-color` custom property takes priority when both are set.

```html
<audio-visualizer color="#10b981"></audio-visualizer>
```

## Transition speed

```html
<!-- Snappy — suits fast speech or music -->
<audio-visualizer style="--audio-visualizer-transition: 40ms"></audio-visualizer>

<!-- Default -->
<audio-visualizer style="--audio-visualizer-transition: 100ms"></audio-visualizer>

<!-- Smooth — suits background ambience -->
<audio-visualizer style="--audio-visualizer-transition: 250ms"></audio-visualizer>
```

## Dark / light theme

The element inherits `currentColor` by default, which means it automatically adapts to your site's text colour:

```css
/* Light theme */
body { color: #111; }

/* Dark theme */
[data-theme='dark'] { color: #f5f5f5; }

/* audio-visualizer bars follow automatically — no extra code */
```
