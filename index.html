Audio Visualizer

A Lit WebComponent that displays real-time microphone frequency bands as animated bars.

Usage

html
<!-- Import the component -->
<script type="module" src="/src/main.ts"></script>

<!-- Place the element -->
<audio-visualizer
  size="md"
  color="#6366f1"
  bar-count="5"
></audio-visualizer>
javascript — start microphone
// Must be called from a user gesture (click) to satisfy
// browser autoplay policy and trigger the permission prompt.
const viz = document.querySelector('audio-visualizer');

button.addEventListener('click', async () => {
  await viz.startMicrophone();
});
javascript — stop microphone
viz.stopMicrophone(); // Releases the mic stream immediately

Styling

CSS custom property
/* preferred — keeps colour in CSS */
audio-visualizer {
  --audio-visualizer-color: #6366f1;
  --audio-visualizer-transition: 60ms;
}
CSS inheritance
/* bars pick up the CSS color property */
<div style="color: #ec4899">
  <audio-visualizer>
  </audio-visualizer>
</div>
HTML attribute
/* quick, inline convenience */
<audio-visualizer
  color="#10b981"
></audio-visualizer>
CSS Custom Property Default Description
--audio-visualizer-color currentColor Bar colour. Takes priority over the color attribute. Falls back to the inherited CSS color property when unset.
--audio-visualizer-transition 100ms Duration of bar height transitions. Lower = snappier response; higher = smoother, slower animation.
html — transition speed
<!-- Snappy -->
<audio-visualizer style="--audio-visualizer-transition: 40ms"></audio-visualizer>

<!-- Smooth -->
<audio-visualizer style="--audio-visualizer-transition: 250ms"></audio-visualizer>

Device Switching

javascript
// Enumerate audio input devices (requires mic permission first)
const devices = await navigator.mediaDevices.enumerateDevices();
const mics = devices.filter(d => d.kind === 'audioinput');

// Pass a deviceId to use a specific microphone.
// Calling startMicrophone() again stops the current stream first.
select.addEventListener('change', async () => {
  await viz.startMicrophone(select.value || undefined);
});

Props

Attribute Type Default Description
size 'icon' | 'sm' | 'md' | 'lg' | 'xl' 'md' Controls bar width, container height, and gap between bars.
color string (any CSS color) inherits Bar fill color. Convenience attribute — prefer --audio-visualizer-color for stylesheet theming.
bar-count number 3 or 5 Number of frequency bands shown. Defaults to 3 for icon/sm, 5 for larger sizes.
Method Signature Description
startMicrophone (deviceId?: string) => Promise<void> Requests mic access and starts the animation. Restarts automatically if already active.
stopMicrophone () => void Stops the animation loop and releases the microphone stream.
